def ease4lmp.lammps_data.LammpsBonds.get_maximum_per_atom (   self)
Returns
The maximum number of topology components per atom.

This method overrides LammpsTopology.get_maximum_per_atom, since data for a bond is linked to the first atom in the bond whereas data for a topology component except for bond is linked to the second atom in the component. See also Lammps source code associated with bonds per atom, Atom::data_bonds().
