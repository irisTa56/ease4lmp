Parameters
atom_styleA string denoting Lammps atom style.
positionsA numpy.ndarray of which shape is (N, 3), where N is the number of atoms. Elements of the array at (i, 0), (i, 1) and (i, 2) are x, y and z coordinates of the i th atom, respectively.
velocitiesA numpy.ndarray of which shape is (N, 3), where N is the number of atoms. Elements of the array at (i, 0), (i, 1) and (i, 2) are velocities of i th atom in the x, y and z direction, respectively.
typesA list of which length is equal to the number of atoms. Each element denotes a type of each atom used in Lammps.
